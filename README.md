# 使用说明
得到本项目之后你需要保证你的电脑拥有以下的程序:
- Python3
- adb

在运行之前需要使用adb与模拟器链接具体代码在main.py,并且拥有以下的库：
- opencv-python

在安装结束之后，运行main.py即可
